Conteudos:



HTML5 é a quinta e mais recente versão da linguagem de marcação de hipertexto (HTML), usada para estruturar o conteúdo de páginas web. Ele introduz novos elementos e funcionalidades que facilitam o desenvolvimento de sites modernos, com foco em interatividade, multimídia e compatibilidade com dispositivos móveis.

CSS, ou Folhas de Estilo em Cascata, é uma linguagem usada para descrever a apresentação de documentos HTML ou XML. Em termos simples, o CSS define como os elementos de uma página web devem ser exibidos, incluindo cores, fontes, layout e outros aspectos visuais. O CSS trabalha em conjunto com o HTML, que define a estrutura do conteúdo, enquanto o CSS adiciona estilo e apresentação.

O que é a propriedade Display em CSS


A propriedade display em CSS controla o comportamento visual de um elemento HTML, determinando como ele interage com outros elementos na página e como ele ocupa espaço. Essencialmente, define se um elemento se comporta como um bloco, um elemento em linha (inline), ou usa layouts mais complexos como flexbox ou grid.

Display Caracteristicas
block O elemento ocupa toda a largura disponível, começando em uma nova linha e forçando outros elementos a irem para a próxima linha.
Inline O elemento ocupa apenas o espaço necessário para seu conteúdo e pode compartilhar a mesma linha com outros elementos inline.
Inline-block O elemento combina o comportamento de inline e block. Ele pode compartilhar a mesma linha com outros elementos, mas também permite definir largura e altura.
Flex Cria um container flexível que organiza seus filhos em uma ou mais linhas/colunas, permitindo layouts flexíveis e responsivos.
Grid Cria um container com grade, dividindo o espaço em linhas e colunas, ideal para layouts complexos e bidimensionais.
